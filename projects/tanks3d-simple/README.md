# Tanks 3D — simple

Сравнительный вариант без development-песочницы и генераторов. Это один
обычный runtime-файл с вручную описанными ареной, танками и волнами. Открой
`index.html` в браузере; управление — `WASD`, мышь для башни, `Space` для
выстрела и `Shift` для турбо-рывка. Карта выбирается в верхней панели.

Цель варианта — не плохой код, а обычная быстрая сборка: при росте контента
правки остаются в одном прямом месте.

Общий для обеих версий V10 brief находится в [BRIEF_V10.md](./BRIEF_V10.md).
