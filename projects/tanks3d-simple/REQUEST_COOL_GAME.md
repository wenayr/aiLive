# Входящий запрос

> Сделай игру классной.

## Решение direct-ветки

Запрос не задаёт метрику или обязательный список механик. Для одного прямого
runtime выбран самый короткий видимый результат: турбо-рывок игрока по `Shift`,
ускорение движения, cooldown в HUD и выхлоп в рендере. Новая игровая сущность не
нужна: это намеренно остаётся локальной механикой `game.js`.
