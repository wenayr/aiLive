# Цель: Tableer

Цель development-песочницы — `../../projects/tableer`.

Сейчас это реальная минимальная площадка, а не искусственная задача: headless
таблица хранит строки, фильтрует их и проецирует эти операции через facade.
Следующая задача должна появляться от продуктовой потребности и менять только
те tracked paths, которые действительно относятся к этой потребности.

Не создавать генератор, mock-generator, MCP или дополнительный слой только
потому, что в проекте есть три файла. Основанием для роста служит повторение,
источник истины и независимая проверка.
