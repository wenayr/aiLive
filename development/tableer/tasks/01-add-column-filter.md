# Первая следующая задача: фильтр по выбранному столбцу

Добавь в Tableer возможность искать не по всем значениям строки, а по явно
выбранному столбцу. Это должна быть реальная небольшая продуктовая операция,
а не демонстрационная симуляция genetic module.

Перед началом:

1. прочитай `../AGENTS.md`, `../TARGET.md` и продуктовый `project.yaml`;
2. выполни `npm run status` или первоначальный `npm run setup`;
3. спланируй минимальную правку и тест в Tableer.

После сохранённой законченной правки:

1. выполни проверку Tableer;
2. выполни `npm run scan` из этой development-песочницы;
3. обработай ровно одно pending action за раз через `memory/response.json` и
   `npm run resolve`;
4. не формируй наблюдение, если одного изменения недостаточно: выбери skip.
