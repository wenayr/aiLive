# V11 visual field test

`field-test.html` — отдельный product-режим, не агент и не новая sandbox.
Он обязан показать один танк, мышь для башни, `W/S` для движения, `A/D` для
корпуса, стрельбу `Space` и полностью разрушаемые блоки.

Нужно десять визуально и структурно разных карт. Решение sandbox-ветки:
существующий `createArena` выбирает profile и palette для каждой карты, а
field runtime превращает возвращённые walls в блоки с hp. Он не копирует
правила карты в runtime. Renderer показывает только живые блоки.

Проверить: десять map kinds, безопасный start, мышь → башня, W/S/A/D → корпус,
снаряд → разрушение блока, browser-smoke обеих field-test страниц.
