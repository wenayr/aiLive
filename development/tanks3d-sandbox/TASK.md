# Итерация 2: развить волну, не расширяя систему вслепую

Игра уже имеет генераторы арены, танка и начальной волны. Добавь одну реальную
возможность: следующая волна должна появляться после зачистки текущей и расти по
round. Источник истины — `round` и spawn pads арены; не копируй список танков в
game runtime.

После сохранения:

1. выполни `npm --prefix ../../projects/tanks3d-sandbox run verify`;
2. выполни `npm run scan`;
3. если discover просит оценку, проверь только изменения wave generator и
   runtime; поскольку это первое наблюдение этой песочницы, честный ответ —
   `skip`, если нет доказанной повторной ошибки;
4. выполни `npm run resolve` и запиши факт в curated evidence.
