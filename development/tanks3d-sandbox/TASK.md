# Итерация 3: карты, модели и прицел

Игра уже имеет генераторы арены, танка и волны. Добавь одну связанную реальную
возможность: выбор map profile, видимые model parameters танка и прицел игрока
в мировых координатах. Источники истины — `mapKind`, design archetype и
`input.aim`; не копируй карты или характеристики танков в game runtime.

После сохранения:

1. выполни `npm --prefix ../../projects/tanks3d-sandbox run verify`;
2. выполни `npm run scan`;
3. если discover просит оценку, проверь только изменения generator contracts и
   runtime; сохрани instruction лишь если видна конкретная повторяющаяся
   инженерная граница, а не просто потому, что изменилось несколько модулей;
4. выполни `npm run resolve` и запиши факт в curated evidence.
