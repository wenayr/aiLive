# Tanks 3D development sandbox — правила агента

Этот каталог — внешний рабочий корень для generator-варианта Tanks 3D. Цель
строго задана в `sandbox.json`: `../../projects/tanks3d-sandbox`. Внутри цели
находится только игровая продуктовая архитектура; agent runtime и память остаются
здесь, в `memory/`.

## Первая сравнительная задача

1. Выполни `npm run status`, а перед первой правкой — `npm run setup`.
2. Меняй только обычный код цели и только для реальной игровой возможности.
3. После законченной сохранённой правки выполни product test, затем `npm run scan`.
4. Если создан `memory/pending.json`, ответь лишь на предложенный узкий вопрос.
   Не сохраняй instruction по одному слабому наблюдению: используй `discover/skip`.

## Границы

- Не помещай агента, memory, pending actions или development-инструкции в игру.
- Генератор оправдан только для реального повторяемого игрового ресурса: арены,
  архетипов танков или волн. Не добавляй генератор ради числа модулей.
- В этом сравнении простая версия сознательно остаётся самостоятельным продуктом
  и не импортируется ни сюда, ни в game target.
