# Evidence

В этот каталог попадают только короткие, воспроизводимые выводы. Живые логи и
состояние mechanics probe находятся в `sandbox/genetic-module/genetic/` и не
коммитятся. Утверждение считается доказанным только вместе с командой проверки
и наблюдаемым результатом; предположения отмечаются отдельно.

- [TANKS3D_COMPARISON.md](./TANKS3D_COMPARISON.md) — первый двухитерационный
  прогон simple browser-product против generator product target.
- [TANKS3D_V10_COMMON_PASS.md](./TANKS3D_V10_COMMON_PASS.md) — один общий
  abstract brief из десяти векторов, выполненный отдельно в обеих версиях.
- [TANKS3D_V10_ASSESSMENT.md](./TANKS3D_V10_ASSESSMENT.md) — оценка того,
  какие части прохода оказались проще, сложнее и структурно выгоднее.
