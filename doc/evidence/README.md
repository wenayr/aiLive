# Evidence

В этот каталог попадают только короткие, воспроизводимые выводы. Живые логи и
состояние эксперимента находятся в `sandbox/genetic-module/genetic/` и не
коммитятся. Утверждение считается доказанным только вместе с командой проверки
и наблюдаемым результатом; предположения отмечаются отдельно.
