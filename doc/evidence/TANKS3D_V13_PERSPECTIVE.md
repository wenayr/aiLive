# Tanks 3D V13: native perspective comparison

Дата: 2026-07-15.

## Задача

Одинаковый абстрактный запрос «сделать игру полностью 3D и разнообразной»
сведён к проверяемому экрану: chase-camera за танком, перспективная проекция,
десять карт, три разных архетипа врагов и разрушаемые объекты с hp.

## Реализация

Обе игры получили независимые `perspective-test.html` страницы. Рендеринг
нативный Canvas: ground tiles и объёмные cuboid-модели проходят через
перспективную проекцию и рисуются по глубине. Внешний 3D-движок намеренно не
добавлен: это сохраняет честность сравнения и не создаёт нерассмотренную
зависимость.

Sandbox-вариант повторно использует generator product code для арены и
archetype tanks; simple-вариант хранит ручные правила непосредственно в своём
runtime. В обоих продуктах нет genetic memory или agent runtime.

## Критерий следующего сравнения

Этот проход показывает только функциональную возможность структуры. Для
сравнения эффективности следующий общий запрос должен измерять отдельно:
время, число затронутых product files, повторные ошибки и provider token-ledger
в момент реальных вызовов модели. Размер файлов не равен фактическим токенам.

## Проверки

- `npm run verify`: прошли genetic module, Tableer, обе development-песочницы,
  simple (3 tests) и sandbox product (14 tests).
- Browser smoke: обе `perspective-test.html?map=canyon` страницы показали
  `perspective 3D`, `hull`, `enemies`, `objects` и не дали console errors.
- В sandbox browser-сцене движение мыши вправо развернуло видимый белый ствол
  вправо от корпуса; тем самым проверены экранный aim и отличная от прежней
  изометрической проекция.
