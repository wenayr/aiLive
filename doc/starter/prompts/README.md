# Операционные режимы

Выбирайте ровно один режим на сессию. Общий master prompt используется только для исследования и передачи контекста; он не заменяет эти инструкции.

| Режим | Назначение | Меняет stable |
|---|---|---:|
| [BUILD](./BUILD.md) | Создать реальный baseline | Да, в согласованной области |
| [SCAN](./SCAN.md) | Восстановить AS-IS | Нет |
| [GENERATOR_HUNT](./GENERATOR_HUNT.md) | Найти генеративные candidates | Нет |
| [ACQUIRE](./ACQUIRE.md) | Исследовать внешние варианты | Нет |
| [EXTRACT](./EXTRACT.md) | Материализовать выбранный candidate | Только candidate scope |
| [REVIEW](./REVIEW.md) | Независимо оценить evidence | Нет |

Права должны обеспечиваться средой. Текст инструкции не является sandbox.
